In this practical we will learn how to validate an HTML5 file (i.e. check that the content is valid and properly formed) using automated (or technical) and non-automated approaches. We will do this by using the World Wide Web Consortium’s (W3C’s) free validation service, which is available online. This checks the validity of the code, and if it finds mistakes, tells you where in the file you have gone wrong. We will also conduct a visual inspection of website source code.
